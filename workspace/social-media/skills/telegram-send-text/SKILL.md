# telegram-send-text Skill

Send a clean user-facing Telegram text message with automatic sanitization.

## Use this workflow

1. Prepare concise user-facing text (2-4 lines).
2. Run:

```bash
node skills/telegram-send-text/scripts/send_text_clean.js --text "Your post is scheduled. Please confirm next step."
```

3. Do not include raw tool output, file paths, command strings, or JSON.
